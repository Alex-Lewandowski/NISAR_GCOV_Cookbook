# Set Up an Earthdata Login Account for NISAR Data Access

<br/>

You must have an Earthdata Login account and accept the `Alaska Satellite Facility Data Access` {abbr}`EULA (End User License Agreement)` to access NISAR data.

## Create an Earthdata Login Account

1. Navigate to Earthdata Login: https://urs.earthdata.nasa.gov/
2. Click the `REGISTER FOR A PROFILE` button.

:::{figure} ../assets/EDL_landing_page.png
:alt: Image of the Earthdata Login landing page

:::

3. Complete the registration form to create an account.
4. Click the emailed link to verify your email address.
5. Log into your new Earthdata Login account

## Accept the `Alaska Satellite Facility Data Access` {abbr}`EULA (End User License Agreement)`

1. Navigate to the `EULAs` tab in your Earthdata Login account.
2. Click the `Accept New EULAs` link.

:::{figure} ../assets/EDL_EULA_tab.png
:alt: Image showing the Earthdata Login Eula tab

:::

3. Scroll to the bottom of the EULA list, find the `Alaska Satellite Facility Data Access` entry and click its `ACCEPT EULA` button.

:::{figure} ../assets/EDL_ASF_EULA.png
:alt: Image showing the "Alaska Satellite Facility Data Access" "ACCEPT EULA" button

:::

4. A EULA will appear. Read it and scroll to the bottom. Agree to the terms by clicking the checkbox.
5. Click the `AGREE` button.
